# Birthday Countdown project
- A simple birthday countdown project built with HTML,CSS and JS

## Features
* Live countdown timer(Days, hours, minutes, seconds)
* Smooth heartbeat animation
* Clean and minimal UI
* Beginner-friendly code

### Top languages
* HTML
* CSS
* JS

* CSS animations
